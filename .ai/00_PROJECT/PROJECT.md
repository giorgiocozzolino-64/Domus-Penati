# Project Overview

Domus Penati is a Family 5.0 digital home platform designed to help families preserve memories, rituals, and emotional connection in a calm and accessible environment. The repository currently contains the foundation of an MVP frontend experience built around this concept.

# Vision

To create a warm, trusted digital home where families can gather stories, memories, and meaningful moments in a way that feels intimate, simple, and lasting.

# Mission

To build an elderly-friendly, human-centered digital experience that makes family memory feel private, meaningful, and easy to share across generations.

# Core Philosophy

- Simplicity over complexity
- Calm and respectful interaction
- Privacy and emotional safety
- One clear action per screen
- Technology that supports belonging rather than distraction
- A gentle experience designed for everyday use

# Target Users

- Families who want to preserve shared memories and traditions
- Older adults seeking a simple and reassuring digital experience
- Younger relatives who want to contribute stories and keep family continuity alive
- Caregivers and households looking for a private, emotionally resonant home platform

# Technology Stack

- Next.js with the App Router
- React and TypeScript
- Tailwind CSS for interface styling
- Supabase for authentication and data persistence
- Server and client-side Supabase integration helpers
- Middleware-based route protection

# Repository Structure

```text
app/                # Main routes and pages for onboarding, authentication, welcome flow, memories, and invitations
components/         # Reusable UI primitives and layout building blocks
lib/                # Copy, authentication actions, and Supabase clients
types/              # Shared TypeScript definitions
middleware.ts       # Route protection and access control
``` 

# Current Status

The repository contains the foundational MVP frontend structure, including onboarding and authentication-related pages, a family welcome experience, memory-focused screens, and an initial UI system. The project is in an early implementation stage and is being shaped around the Domus Penati product vision.

# Active Sprint

The current focus is to turn the concept into a coherent MVP experience by refining the core user journey, strengthening the gentle interface language, and aligning the frontend structure with Supabase-backed data flows.

# Long-term Goal

To evolve Domus Penati into a complete Family 5.0 platform that helps households preserve their history, strengthen intergenerational bonds, and keep shared memories alive in a serene digital home.
